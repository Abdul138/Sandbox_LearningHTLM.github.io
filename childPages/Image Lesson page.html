Steps in building an Image
  • Pick an Image (cereful of the file extention in the url)
  • Record the URL, cite it, comment for the date accessed, and record the original dimensions
  • Note: opening in a new tab will eventually provide a http or https URL with a file extension
  • Use blockquotes and comments as appropriate
  • Use img tags with attributes src, alt, title, and style with width-height
  • Compare the same image with two sizes of the viewport (25% & 100%)
